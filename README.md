# Sterling Tutoring Group — Landing Page

The official marketing landing page for [Sterling Tutoring Group](mailto:info@sterlingtutoringgroup.com.au) — bespoke one-on-one and small group tutoring for Year 7–12 students across Australia.

## About

Sterling Tutoring Group pairs high school students with a dedicated tutor who's already dominated the system they're navigating. Every tutor on our panel scored 90+ in their ATAR and is currently studying in some of Australia's most competitive degrees — Law, Medicine, and Engineering.

Our core values: **integrity, simplicity, proficiency, and being second to none.**

## Running locally

This is a static HTML site with no build step. To preview locally:

```bash
python3 -m http.server 8765
```

Then open [http://localhost:8765](http://localhost:8765) in your browser.

## Deployment

Deployed via [Vercel](https://vercel.com) from this repository.
